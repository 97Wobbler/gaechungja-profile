# 개청자 아이콘 생성기

[![English](https://img.shields.io/badge/lang-English-blue)](./README.en.md)

> 🎮 **[지금 바로 사용해보기](https://97wobbler.github.io/gaechungja-profile/)**

랜덤 개청자 아이콘을 생성하고 다운로드할 수 있는 웹 앱입니다.

---

## 기능

- 랜덤 캐릭터 생성
- 캐릭터 이미지 다운로드
- 등급별 통계 표시 (SSSS ~ N)
- 희귀도 등급 계산 및 표시
- 등급별 효과음 재생
- 다국어 지원 (한국어/영어)

---

## 사용법

1. **생성** 버튼을 클릭하여 랜덤 캐릭터를 생성합니다.
2. **다운로드** 버튼을 클릭하여 PNG 파일로 다운로드합니다.

---

## 기술 스택

- HTML5 Canvas
- Vanilla JavaScript (ES Modules)
- CSS3

## 로컬 개발 서버

```bash
npx serve
```
