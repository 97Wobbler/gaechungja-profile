A web app for generating and downloading random Gaechungja icons.

---

## Features

- Random character generation
- Character image download
- Grade statistics (SSSS ~ N)
- Rarity grade calculation
- Grade-based sound effects
- Multi-language support (Korean/English)

---

## Usage

1. Click **Generate** to create a random character.
2. Click **Download** to save as a PNG file.

---

## Tech Stack

- HTML5 Canvas
- Vanilla JavaScript (ES Modules)
- CSS3
